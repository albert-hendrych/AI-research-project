# Deep Dive into Artificial Intelligence: NLP Joke Generation Model

## 📝 Overview
This repository contains the practical and theoretical components of my 50-page research project on Artificial Intelligence, conducted prior to the widespread rise of large-scale consumer models like ChatGPT. The project explores the internal mechanics of neural networks and the evolution of Natural Language Processing (NLP).

## 🚀 Key Features
* **Custom NLP Model:** Developed a generative language model using Python, specifically trained on a curated dataset of jokes to explore text pattern recognition and creativity in AI.
* **Comprehensive Research:** An extensive 50-page technical paper analyzing AI architectures, training methodologies, and future implications of generative systems.
* **Implementation:** Developed entirely in Python, focusing on data cleaning, tokenization, and model training.

## 🛠️ Tech Stack
* **Language:** Python
* **Field:** Natural Language Processing (NLP) / Deep Learning
